HTML:

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web, including text, images, links, and more.

Basic HTML Structure

<!DOCTYPE html>
<html>
<head>
   <title>Document</title>
</head>

<body>

<!-- All the content of Webpage written here. -->
</body>
</html>
  • !DOCTYPE html- Defines document type as HTML5
  • html- Root element of the HTML document.
  • head- Contains meta, title, and links
  • title- Sets the page title shown in browser tab
  • body- Contains all visible content of the page

HTML elements

An HTML element is a complete set that consists of a start tag or (opening tag), content, and an end tag or (closing tag).

Heading tag - <h1> to <h6>

Paragraph tag - <p>

Anchor tag - <a>

Unorderd list - <ul>

Ordered list - <ol>

Break tag - <br>

Horizontal rule/line - <hr>

Table tag - <table>

Marquee tag - <marquee>

Anchor Tag

The <a> (anchor) tag is used to create hyperlinks that navigate to another webpage, file, or location. Here is an example.

<a href="https://google.com" target=_blank> Google </a> Google

href="" - We will provide the links of webpage, file, or location where we want to redirect the user.

target=_blank - is used to open the webpage, file, or location in another tab.

Unordered list

An <ul> displays items with bullet points, often used to group related items without a specific order or sequence.

<ul>
    <li>Mango</li>
    <li>Apple</li>
    <li>Grapes</li>
    <li>Lichi</li>
    <li>Watermelon</li>
</ul>
  • Mango
  • Apple
  • Grapes
  • Lichi
  • Watermelon

Ordered list

An <ol> displays items in a specific sequence using numbers or letters, showing a clear, structured order.

<ol>
    <li>Mango</li>
    <li>Apple</li>
    <li>Grapes</li>
    <li>Lichi</li>
    <li>Watermelon</li>
</ol>
  1. Mango
  2. Apple
  3. Grapes
  4. Lichi
  5. Watermelon

Table Tag

The <table> tag in HTML creates a table to organize data into rows and columns using <tr>, <th> and <td>.

<table border="1px" cellpadding= "10px">
   <tr>
     <th>First Name</th>
     <th>Last Name</th>
     <th>Class</th>
     <th>Marks</th>
   </tr>
   <tr>
     <td>Aryan</td>
     <td>Kumar</td>
     <td>6</td>
     <td>78.8%</td>
   </tr>
   <tr>
     <td>Suraj</td>
     <td>Kumar</td>
     <td>6</td>
     <td>80.8%</td>
   </tr>
   <tr>
     <td>Suman</td>
     <td>Kumari</td>
     <td>6</td>
     <td>95%</td>
  </tr>
</table>

Suraj Kumar 6 80.8%

<tr> - tag in HTML defines a table row, grouping together cells created with <td> or <th> inside a table.

<td> - tag in HTML defines a standard data cell in a table row, used to display content within columns.

<th> - tag defines a table header cell, usually bold and centered, representing a title for a column or row.
